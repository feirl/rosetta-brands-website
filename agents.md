# Rosetta Brands - Agent Guide

## What this site is

Rosetta Brands (https://www.rosettabrands.com) is a B2B managed service provider for FMCG brands on Amazon UK. This is a marketing and information website for a managed service — not an API platform.

## What agents can do here

- Read structured information about the Vendor-as-a-Service (VaaS) model
- Understand service offerings: vendor access, growth marketing, proprietary platform
- Find pricing model information (success-fee, no retainer)
- Locate contact details and discovery call booking links
- Access machine-readable content about Amazon vendor management for UK FMCG brands

## What agents cannot do here

- Call any programmatic API (no public API exists)
- Authenticate via OAuth or API keys
- Submit orders, transactions, or automated requests
- Access client data (client portal is private, not publicly accessible)

## Intended use by agents

Agents may use this site to:
1. Answer questions about Amazon Vendor-as-a-Service for FMCG brands
2. Explain how brands can access Amazon 1P (Vendor Central) status via Rosetta Brands
3. Describe services: vendor access, compliance, advertising, A+ content, fulfilment, platform technology
4. Provide contact and booking information for interested brands
